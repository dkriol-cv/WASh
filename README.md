# WASh - Curso LMS "Escola Amiga do WASH"

Este repositório contém o código-fonte do curso interativo "Escola Amiga do WASH", desenvolvido em React + Vite.

## Tecnologias
- React 19
- Vite
- Tailwind CSS
- Framer Motion (Animações)
- Lucide React (Ícones)
- Scorm-again (Integração LMS)

## Como rodar localmente
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
3. Execute o build de produção: `npm run build`
